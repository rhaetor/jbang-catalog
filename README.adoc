= JBang Catalog for Red Hat Build of Apache Camel

For more information about Camel CLI and how to install this catalog, see the Red Hat build of Apache Camel product documentation:
* https://docs.redhat.com/en/documentation/red_hat_build_of_apache_camel/latest/html/tooling_guide_for_red_hat_build_of_apache_camel/index[Tooling Guide for Red Hat build of Apache Camel]

For more information about JBang Catalogs, see the JBang documentation:
* https://www.jbang.dev/documentation/jbang/latest/alias_catalogs.html[Aliases & Catalogs]
* https://www.jbang.dev/documentation/jbang/latest/app-installation.html#installing-from-aliases[Installing Scripts as Apps from Catalog]

== Branches in this repository

When you install the Camel JBang Catalog, use `.x` branches (such as `4.14.x`) 
for the corresponding versions of Red Hat Build of Apache Camel.
You can use other branches if they are recommended in Red Hat Build of Apache Camel documentation.

== Update policy

The versions of Red Hat Build of Apache Camel for Quarkus `.x` branches (such as `4.14.x`)
of the Catalog are updated daily using information from
https://registry.quarkus.redhat.com/client/platforms

After publishing in Quarkus registry, there may be a delay of up to 24 hours
before the versions in this repository are updated 
and a new `.x` branch is created.
